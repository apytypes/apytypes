Construct a new APyFixedArray tensor object with the same shape and fixed-point values as *this, but with a new word-length. The underlying bit-pattern of each tensor element are copied into place, meaning that lowering the number of fractional bits may result in quantization, and lowering the number of integer bits may result in overflowing. Supports quantization and oveflow options on narrowing casts.

bool is_identical(const APyFixedArray &other) const#

Test if *this is identical to another APyFixedArray. Two APyFixedArray objects are considered identical if, and only if:

  • They represent exatly the same tensor shape

  • They store the exact same fixed-point values in all tensor elements

  • They have the exact same bit format (bits, int_bits, and frac_bits)
