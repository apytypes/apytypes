Footnotes

[1] (1,2,3)

numfi is limited to 64 total bits (including results of e.g. multiplication).

[2] (1,2,3,4,5)

pyfixp is essentially a package to simply quantize NumPy-arrays with doubles.

[3]

fpbinary provides a helper function to convert an array to a list with fixed-point numbers.

[4]

Note that this holds for intermediate results as well. For example, multiplying two 27-bit numbers will create a 54-bit result which will not give bit-accurate results with pyfixp.
